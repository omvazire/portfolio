<h2>🚀 About the Project</h2>
<p>
This is my <b>modern developer portfolio website</b> built using <b>React and Vite</b>, designed to showcase my projects, skills, certifications, and experience in an interactive and visually engaging way.
</p>

<p>
The portfolio focuses on <b>clean UI, smooth performance, and responsive design</b>, giving recruiters a quick and impactful overview of my work.
</p>

<hr/>

<h2>✨ Key Features</h2>

<ul>
  <li>⚡ <b>React-Based Architecture</b> – Component-driven scalable structure</li>
  <li>🎨 <b>Modern UI & Animations</b> – Custom effects like Aurora & Bento Grid</li>
  <li>📱 <b>Fully Responsive</b> – Works across all devices</li>
  <li>🧠 <b>Interactive Sections</b> – Hero, About, Projects, Certificates, Contact</li>
  <li>📂 <b>Dynamic Assets</b> – Efficient handling of images, resume & certificates</li>
  <li>🚀 <b>Fast Performance</b> – Powered by Vite</li>
</ul>
